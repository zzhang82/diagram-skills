# Route Comparison - Agent Memory Bridge

## Routes evaluated

### 1. Image-model presentation route
Strengths:
- closest to the reference concept
- strongest composition
- best whitespace and explainer balance
- most premium / editorial feel

Weaknesses:
- less deterministic
- harder to edit structurally

### 2. Structured React/SVG route
Strengths:
- repeatable
- inspectable
- downloadable HTML/SVG/PNG
- best editable fallback

Weaknesses:
- easier to become mechanically styled
- lower visual ceiling without stronger art direction

### 3. Mermaid portable route
Strengths:
- easiest compatibility handoff
- lightweight source
- useful for docs and tool interop

Weaknesses:
- lowest visual ceiling
- not suitable as the default polished presentation route

## Conclusion

For the current skill direction:

- **default presentation route** should be image-model-first
- **editable route** should be React/SVG
- **portable route** should be Mermaid
